## Color Scheme

- **Background Gradient**: Blue (#1e3c72 to #2a5298)
- **Main Content**: White background with rounded corners
- **Primary Action**: Blue gradient buttons
- **Secondary Actions**: Gray buttons
- **Success**: Green (#28a745)
- **Error**: Red (#dc3545)
- **Warning**: Yellow (#ffc107)
- **Info**: Blue (#007bff)

## Font & Typography

- **Font Family**: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- **Header Size**: 3em
- **Subtitle**: 1.2em
- **Body Text**: 1em
- **Card Display**: 1.5em (bold)

## Interactive Elements

1. **Dropdown Menus**: Card selection with suit symbols
2. **Buttons**: Hover effects with subtle lift animation
3. **Tabs**: Active tab highlighted with blue underline
4. **Tables**: Color-coded cells based on action type
5. **Notifications**: Slide-in animation from right, auto-dismiss after 3s
6. **Loading Spinner**: Rotating blue circle during OCR processing

## Responsive Design

- Minimum window size: 800x600
- Default size: 1200x800
- Supports window resizing
- Grid layout adjusts on smaller screens
- Tables scroll horizontally if needed

## Accessibility

- Clear contrast ratios
- Large, readable text
- Descriptive labels
- Keyboard navigable (tab order)
- Color-blind friendly (uses symbols + colors)
