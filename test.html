Blackjack Helper Extension Test Page

Welcome!

This is a test page to demonstrate the Blackjack Helper browser extension.

Instructions:

  1. Install the Blackjack Helper extension in Chrome
  2. The overlay should appear automatically on this page in the top-right corner
  3. Click the extension icon in the toolbar to open the popup
  4. Use the "Toggle Overlay" button to show/hide the overlay
  5. Click the × button on the overlay to close it

Extension Features

  • ✅ Automatic overlay injection
  • ✅ Toggleable visibility
  • ✅ Modern UI design with tabs
  • ✅ Manual card input
  • ✅ OCR card recognition (Tesseract.js)
  • ✅ Basic blackjack strategy recommendations
  • ✅ Odds calculation (win/push/loss/dealer bust)
  • ✅ Expected value (EV) display
  • ✅ Hand analysis (soft/hard/pair detection)

How to Use

Manual Input Mode:

  1. Enter your cards (e.g., "K,7" or "10,A")
  2. Enter dealer's card (e.g., "7")
  3. Click "Calculate" to get recommendations

OCR Recognition Mode:

  1. Click "OCR Recognition" tab
  2. Upload a card image
  3. Click "Process Image"
  4. Use detected cards for calculation

Strategy Recommendations

The extension provides optimal actions based on basic blackjack strategy:

  • HIT - Take another card
  • STAND - Keep your current hand
  • DOUBLE - Double your bet and take one card
  • SPLIT - Split pairs into two hands
